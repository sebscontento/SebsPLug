SebsPlug a stutter plugin made for VCV Rack. 

check out VCV rack 
https://vcvrack.com/

Contributions to this plugin are very welcome!

Use this plugin as a lightweight Stutter effect. 
Use this plugin in combination with MIXXX and VCV rack.
